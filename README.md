- APT 평면도를 이용한 간이 측량
- 간이 측량을 이용한 간이 견적서
## 핵심 기능  Key Feature
- 검색한 아파트의 평면도를 출력.
- 해당 평면도를 CAD 등의 프로그램에 접목시켜 간이 측량.
## 사용 How To Use
- APT 검색하면 평형정보와 평면도를 확인할수 있고, 해당 평면도를 이용하여 인테리어에 필요한 측량을 자동(CAD)으로 해준다
